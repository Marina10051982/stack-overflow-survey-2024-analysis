# Stack Overflow Survey 2024 Analysis

## 📊 Project Overview
This project analyzes the Stack Overflow Developer Survey 2024 dataset to explore trends in developer experience, salaries, and technologies.

## 🛠 Tools
- Python
- Pandas
- Jupyter Notebook

## 📈 Analysis Includes
- Data cleaning and missing values analysis
- Experience distribution
- Salary analysis by country
- Remote work trends
- Python popularity analysis

## 📂 Project File
See the notebook: `stack-overflow-survey-analysis.ipynb`
